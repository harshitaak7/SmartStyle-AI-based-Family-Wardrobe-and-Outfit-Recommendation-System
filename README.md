#SmartStyle – AI-based Family Wardrobe and Outfit Recommendation System

SmartStyle is a full-stack web application designed to help families organize their wardrobe digitally and generate suitable outfit recommendations. The system supports OTP-based email authentication, family profile management, cloud-based wardrobe storage, solo and couple outfit generation, and lookbook saving. It is built using HTML, CSS, JavaScript, and Supabase, making it a secure, scalable, and user-friendly solution for managing clothes and reducing outfit confusion.

Features
OTP-based email authentication.
Family wardrobe management.
Add clothing items with images and metadata.
Cloud storage for wardrobe images.
Solo outfit generation.
Couple outfit generation.
Lookbook for saving preferred outfits.
Migration from localStorage to cloud storage.
Responsive and interactive frontend.

Tech Stack
Frontend: HTML, CSS, JavaScript.
Backend: Supabase.
Database: PostgreSQL.
Authentication: Supabase Auth with OTP login.
Storage: Supabase Storage.

Project Structure
frontend/ – Contains UI components, pages, and styling.
backend/ – Contains Supabase integration and business logic.
assets/ – Stores images and static files.
screenshots/ – Contains project screenshots for documentation.

How It Works
The user logs in using OTP-based email authentication.
The user creates or accesses a family wardrobe.
Clothing items are uploaded with images and details.
The system stores wardrobe data in Supabase PostgreSQL.
Outfit combinations are generated for solo or couple use.
Users can save preferred outfits in the lookbook.
Clothing images are stored securely in Supabase Storage.


Future Scope
AI-based advanced outfit recommendation.
Mobile application version.
Weather-based styling suggestions.
Virtual try-on integration.
Improved recommendation scoring.

Conclusion
SmartStyle provides a practical digital solution for family wardrobe management and outfit planning. It combines a simple user interface with cloud-based backend services to create a modern, reliable, and scalable fashion recommendation platform.
